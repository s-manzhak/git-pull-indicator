# Contributing

## Setup

```bash
git clone https://github.com/s-manzhak/git-pull-indicator.git
cd git-pull-indicator
npm install
npm run compile
```

## Run locally

```bash
npm run install:local
```

Reload the editor window, then open a folder that contains Git repositories.

Or press **F5** with the `Run Extension` launch configuration (edit `debug-test.code-workspace` to point at your test folder).

## Pull requests

1. Fork the repository
2. Create a feature branch
3. Run `npm run compile` before submitting
4. Update `CHANGELOG.md` for user-visible changes
